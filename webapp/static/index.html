Tonality Route

Shortest path between two tonalities on the 24-key D/S/R/P lattice. Works offline — pure JavaScript.
D = dominant (+5th) • S = subdominant (−5th) • R = relative (same key sig) • P = parallel (same root)
Examples: C → a, Eb → f#, F → C, G → em
